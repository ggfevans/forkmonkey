#### Quick Start
```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/forkMonkey.git
cd forkMonkey

# Install dependencies
pip install -r requirements.txt

# Initialize a test monkey
python src/cli.py init

# Run the web UI locally
./start_web.sh

# Run tests
pytest tests/
```
